🧠 Spring Boot Estudos
Estudos práticos de Java com Spring Boot, desenvolvidos durante o curso introdutório da Rocketseat.
O projeto tem como objetivo aplicar conceitos fundamentais de desenvolvimento Back‑End, explorando Spring Security, autenticação com JWT, integração com banco de dados e boas práticas de código.

🚀 Tecnologias utilizadas
Java 17

Spring Boot

Spring Security

Maven

MySQL

JWT (JSON Web Token)

VS Code com extensões para cliente REST (Insomnia, Postman ou API Dog)

📚 Conteúdo estudado
Durante o curso, foram abordados:

Criação de uma API REST para controle de tarefas (To‑Do List).

Implementação de rotas HTTP e validação de parâmetros.

Integração com banco de dados relacional.

Configuração de autenticação e autorização com Spring Security e JWT.

Deploy da aplicação na plataforma Render.

⚙️ Como executar o projeto
Clone este repositório:

bash
git clone https://github.com/moisegit528/Spring-Boot-Estudos.git
Configure o banco de dados no arquivo application.properties.

Execute o projeto:

bash
mvn spring-boot:run
Acesse a API localmente em http://localhost:8080.
